# 社会保険加入フローチャート

以下は、社会保険加入の判定フローチャートです。

```mermaid
graph TD
    A[スタート] --> B{勤務先は社会保険の適用事業所？};

    B -- いいえ --> Z[社会保険加入の対象外];
    B -- はい --> C{週の所定労働時間は20時間以上？};

    C -- いいえ --> Z;
    C -- はい --> D{月額賃金は8.8万円以上？};

    D -- いいえ --> D_Note(※2026年10月以降、この要件は撤廃予定);
    D -- いいえ --> Z;
    D -- はい --> E{雇用期間の見込みは2ヶ月以上？};

    E -- いいえ --> Z;
    E -- はい --> F{あなたは学生ではない？};

    F -- いいえ --> F_Note(※夜間・通信制の学生は加入対象となる場合あり);
    F -- いいえ --> Z;
    F -- はい --> G{勤務先の従業員数は51人以上？};

    G -- いいえ --> G_Note(※2027年10月以降、この要件は撤廃予定);
    G -- いいえ --> Z;
    G -- はい --> H[社会保険に加入];

    Z[社会保険加入の対象外];
    H[社会保険に加入];

    subgraph 補足情報
        D_Note
        F_Note
        G_Note
    end
